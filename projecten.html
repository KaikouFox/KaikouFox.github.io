Serious game in tic80

Ik heb samen met david een serious game gemaakt over het menselijk lichaam.

Je bent een witte bloedcel en je moet alle virussen uit het lichaam halen.

We hebben het gemaakt in tic80, Tic 80 is een retro fantasy console,

dus alles zit er in gebouwd: code editor, sprite editor, map editor, sound editor.

De programeertaal is lua.
